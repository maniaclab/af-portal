Configure a Jupyter notebook (instructions)
Instructions
  1. Please provide a notebook name that has no whitespace.
  2. Please use characters from the set [a-zA-Z0-9._-]
  3. Please use 30 characters or less to name your notebook.
  4. You can request 1 to 16 CPU cores.
  5. You can request 1 to 32 GB of memory.
  6. You can request 0 to 7 GPU instances.
  7. You can select a GPU model based on its memory size.
  8. If you request a GPU, please make sure the GPU is available, by clicking on the icon next to GPU memory.
  9. You can give the notebook a duration of up to 72 hours.
  10. You can choose any Docker image from the dropdown.

Selecting GPU instances

The AF cluster has a variety of GPUs you can choose from. Each GPU can be partitioned into instances. You can have many instances running in parallel.

Please select the number of GPU instances you would like to request. You can select any number from 0 to 4.

Selecting a Docker image

Users can choose between two images: one with full anaconda (ivukotic/ml_platform:conda) and one with NVidia GPU and ROOT support (ivukotic/ml_platform:latest).

The latter has most of the ML packages (Tensorflow, Keras, ScikitLearn, etc.) preinstalled, and a small tutorial with code examples in /ML_platform_tests/tutorial.
